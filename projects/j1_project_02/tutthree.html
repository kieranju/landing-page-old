.toggleClass( )

.toggleClass( ) is a jQuery function that is used to toggle single/multiple classes on an element.


Click the .toggleClass( ) image above to see it in action.

How does it work?

.toggleClass( ) is as simple as it sounds. You must pass in a parameter of whichever class you wish to toggle; for example, .toggleClass("foobar") to toggle the class "foobar".

HTML: 
<div id="objectToggle"></div>
CSS: 
#objectToggle {
    height: 100px;
    width: 100px;
    background-color: slategray;
    border-radius: 50%;
}
JQUERY: 
var objectToggle = $("#objectToggle");

objectToggle.click(function() {
    objectToggle.toggleClass("foobar");
});

In the example used on this page, I have selected the element with an id of "objectToggle" and stored it in a variable named objectToggle.


For the purpose of this example, a class will be toggled whenever said object is clicked. We toggle the class "foobar", and within our CSS, we can provide properties for any element with the class "foobar".

CSS: 
.foobar {
    box-shadow: 0px 0px 5px 10px rgba(255,255,255,0.3);
}
