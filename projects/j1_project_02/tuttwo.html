.keypress( )

.keypress( ) is a jQuery function that can bind an event handler to the keypress JavaScript event.


Press the "D" key on your keyboard to see it in action.

How does it work?

.keypress( ) can be used on much more than just page content, as it allows you to perform actions to modify any piece of content on your webpage at the press of a keyboard key.

JQUERY: 
$(document).keypress(function(e) {
    if(e.which == 100) 
    {
        // Code to be executed
    }
});

Here, the integer that it watches for is the keyCode registered to any specific key. In this case, 100 is the "d" key on your keyboard.


For reference, this tool is great for determining keyCode values.


Depending on what you want to use this for, you can replace in which object it will be listening for a keypress. Here, I use the entire webpage; however, it is possible to restrict this to a single input, Ã  la textboxes.
