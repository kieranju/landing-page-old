JavaScript/JQuery Tutorials

JavaScript? What's that?

JavaScript is an object-oriented computer programming language commonly used to create interactive effects within web browsers; however, be warned. Don't let its short lived description form any conclusions about how great it is to work with. JavaScript, as realized by few, lucid, well-disposed developers, is a nightmare.


JQuery takes pride in solving this nightmare. JQuery is a multi-browser JavaScript library designed to counter the hellish properties of JavaScript on its own. Using JQuery will make life easier, so use it.
