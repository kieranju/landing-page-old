.animate( )

.animate( ) is a jQuery function that allows for the animation of numeric CSS changes to objects on a web page.


Click the .animate( ) image above to see it in action.

How does it work?

HTML: 
<div id="objectAnimate"></div>
CSS: 
#objectAnimate { 
    height: 100px;
    width: 100px;
    background-color: slategray;
    border-radius: 50%;
}

We first generate an element and style it whichever way suits your fancy. For this example, I have created a small, blue-gray circle.


Next, we must create a way to execute the animate function. The following allows us to execute code whenever the element we have created is clicked.

JQUERY: 
var objectAnimate = $("#objectAnimate");

objectAnimate.click(function() {

    // Code to be executed when clicked

});

It is worthy to note that if this script is loaded before the document/page is loaded, it will not work. This is due to lack of the presence of our element "objectAnimate" before it selects it in our script.

JQUERY: 
var isAnimate = false;

function animate() {
    if (isAnimate == false) {
        isAnimate = true;
        objectAnimate.animate({"left":"25px"}, 150).animate({"left":"0px"}, 250, function() {
            isAnimate = false;
        });
    }
}

animateObject.click(function() {
    animate();
});

To keep code clean, it will be a good idea to create a function to handle the animation. Here, I create a function named "animate" and call it within the click event of our object.


I have added a new variable "isAnimate". This is to prevent the animation from queueing if you click the object while the animation is playing.


Clicking on our circle should now yield in some form of animated movement.
